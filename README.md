### User Preferences

SecV supports user preferences for customization:

**Location:** Settings are stored in `~/.secv/` (auto-created)

**Example Preferences:**
```json
{
  "default_output": "json",
  "color_output": true,
  "verbose": false,
  "timeout": 300,
  "concurrent_modules": 5,
  "auto_update": true,
  "update_interval": 24
}
```

### Environment Variables

```bash
# Set default installation tier
export SECV_INSTALL_TIER=full

# Set custom tools directory
export SECV_TOOLS_DIR=/custom/path/tools

# Set custom cache directory
export SECV_CACHE_DIR=/custom/path/cache

# Disable auto-updates
export SECV_AUTO_UPDATE=false

# Set proxy for updates
export SECV_PROXY=http://proxy:8080
```

### Module Configuration

Modules can have configuration files in `.secv/`:

```bash
# Example: ~/.secv/portscan.conf
[scanner]
default_engine = syn
default_ports = top-100
timeout = 60
threads = 50

[output]
format = json
save_pcap = true
```

### API Integration

Use SecV programmatically:

```python
#!/usr/bin/env python3
import subprocess
import json

def run_secv_module(module, target, params=None):
    """Execute SecV module programmatically"""
    context = {
        "target": target,
        "params": params or {}
    }
    
    cmd = [
        "./secV",
        "-m", module,
        "-t", target
    ]
    
    if params:
        for key, value in params.items():
            cmd.extend(["-p", f"{key}={value}"])
    
    result = subprocess.run(
        cmd,
        capture_output=True,
        text=True
    )
    
    return json.loads(result.stdout)

# Example usage
result = run_secv_module(
    module="portscan",
    target="example.com",
    params={"ports": "web", "engine": "syn"}
)

print(f"Found {len(result['data']['open_ports'])} open ports")
```

## Ã°Å¸â€º Ã¯Â¸ Troubleshooting

### Common Issues

**Issue: Module not found**
```bash
# Solution 1: Reload modules
secV > reload

# Solution 2: Check module directory
ls -la tools/

# Solution 3: Verify module.json syntax
jq empty tools/category/module/module.json
```

**Issue: Permission denied**
```bash
# Solution: Make files executable
chmod +x secV
chmod +x install.sh
chmod +x tools/*/module.py
```

**Issue: Dependencies missing**
```bash
# Solution: Reinstall dependencies
pip3 install -r requirements.txt --user --break-system-packages

# Or run installer again
./install.sh
```

**Issue: Go binary not compiling**
```bash
# Solution: Install Go
# Ubuntu/Debian
sudo apt install golang-go

# Arch
sudo pacman -S go

# macOS
brew install go

# Then recompile
cd SecV
go build -o secV main.go
```

**Issue: Update fails**
```bash
# Solution 1: Check git status
git status
git stash  # Save local changes
git pull
git stash pop

# Solution 2: Rollback
python3 update.py --rollback

# Solution 3: Repair installation
python3 update.py --repair

# Solution 4: Clean install
cd ..
git clone https://github.com/SecVulnHub/SecV.git SecV-new
cd SecV-new
./install.sh
```

**Issue: ADB not detecting device (Android modules)**
```bash
# Solution: Restart ADB
adb kill-server
adb start-server
adb devices

# Enable USB debugging on device
# Settings > Developer Options > USB Debugging

# Accept authorization on device
```

**Issue: Scapy requires root**
```bash
# Solution: Use sudo for raw packet operations
sudo ./secV
secV > use portscan
secV (portscan) > set engine syn
secV (portscan) > run target.com
```

**Issue: Slow module execution**
```bash
# Solution: Increase timeout
secV (module) > set timeout 600

# Or reduce concurrency
secV (module) > set threads 10
```

### Debug Mode

Enable verbose output:

```bash
# Start SecV with debug flag
./secV --debug

# Or set in module
secV (module) > set verbose true
secV (module) > run target.com
```

### Graceful Degradation Pattern

Make your modules work across all installation tiers:

```python
#!/usr/bin/env python3
import json
import sys
import socket  # stdlib - always available

# Optional dependency detection
try:
    import scapy.all as scapy
    HAS_SCAPY = True
except ImportError:
    HAS_SCAPY = False

try:
    import requests
    HAS_REQUESTS = True
except ImportError:
    HAS_REQUESTS = False

def scan_advanced(target):
    """Advanced scan with scapy"""
    if not HAS_SCAPY:
        return None
    # Use scapy features
    return {"method": "syn", "data": "..."}

def scan_basic(target):
    """Basic scan with stdlib"""
    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    # Basic implementation
    return {"method": "connect", "data": "..."}

def main():
    context = json.loads(sys.stdin.read())
    target = context['target']
    
    # Inform about capabilities
    if not HAS_SCAPY:
        print("INFO: Install scapy for SYN scan: pip3 install scapy", 
              file=sys.stderr)
    
    # Choose best method
    if HAS_SCAPY:
        result = scan_advanced(target)
    else:
        result = scan_basic(target)
    
    print(json.dumps({"success": True, "data": result}))

if __name__ == '__main__':
    main()
```
